Q-NumericalRangeChecker DPU measures the number of properties that have the objects in a specified range. 

  • Input: RDF graph. To extract such information you may use (E-HttpDownload->T-UnZipper->T-FilesFilter->T-FilesToRDF->Q-NumericalRangeChecker)
  • Output: the quality output is an RDF named graph based on DaQ vocabulary which in turn is given as input to the T-RdfToFiles DPU
