Q-CompletenessChecker DPU measures the number of missing elements from a specified list of subject/property. 

  • Input: RDF graph. To extract such information you may use (E-HttpDownload->T-UnZipper->T-FilesFilter->T-FilesToRDF->Q-CompletenessChecker)
  • Output: the quality output is an RDF named graph based on DaQ vocabulary which in turn is given as input to the T-RdfToFiles DPU
